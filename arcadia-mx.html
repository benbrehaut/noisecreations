ARCADIA-MX

11 Themes | 8-Bit, Retro and Modernised Game Music.

arcadia-mx demo

Launching ARCADIA-MX – a modernised 8-bit-inspired musical collection! This collection has been composed by Ollie Weikert. Keeping true to the modernised 8-bit/retro style of the ARCADIA SFX collection, Ollie has expertly created 11 musical themes to be used in a wide variety of game types. Each theme consists of 4 distinct states (average duration ~1 minute each) – ‘Base’, ‘Stealth’, ‘Intense’ and ‘Boss’, so you can match the mood of an in-game situation whilst maintaining the same core theme. Instrument stems and some extra musical components such as cymbal swells and percussion elements are also included for maximum fidelity. The themes are as follows: INNER CITY, JUNGLE, FOREST, WILDERNESS, DESERT, CASTLE, NIGHT TIME, SPY, UNDERWATER, FIELD and RACING.

KEY FEATURES

  • • 410+ 48 kHz, 24-bit / stereo BWAV files
  • • 11 themes, each with 4 discrete states + instrument stems + extra musical components
  • • 9.5 GB of data
  • • All audio includes embedded descriptive metadata
